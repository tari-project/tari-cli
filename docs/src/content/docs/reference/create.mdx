title: tari create

---

Creates a new workspace for your Tari template project.

```bash
tari create <name> [OPTIONS]
```

**Alias:** `tari new`

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Project name (converted to snake_case) |

## Options

| Option | Description |
|--------|-------------|
| `-t, --template <ID>` | Project template ID (prompted if not set) |
| `-o, --output <PATH>` | Output directory (default: current directory) |
| `--skip-init` | Skip git initialization |
| `--skip-metadata` | Skip automatic metadata initialization |
| `-v, --verbose` | Enable verbose output |

## Examples

```bash
# Interactive - prompts for template selection
tari create my-project

# Specify template directly
tari create my-project -t basic

# Custom output directory
tari create my-project -o ~/projects
```

## What it does

1. Scaffolds a new Cargo workspace using a project template
2. Creates a `tari.config.toml` with default network configuration
3. Initializes WASM template crates (if defined by the project template)
4. Sets up metadata generation (`build.rs` + `[package.metadata.tari-template]`) on each template crate
5. Initializes a git repository
